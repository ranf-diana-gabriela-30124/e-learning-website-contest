Capitolul 9 - tablouri uni si bidimensionale
  • Tablouri unidimensionale

    • Un tablou unidimensional reprezinta o multime / un sir de valori care au acelasi tip de date ( int, char, double etc.). In urmatoarele paragrafe veti afla mai multe detalii despre declararea unui astfel de tablou si beneficiile sale.


  • Declararea unui tablou unidimensional

    • tipDeBază denumire[Dimensiune];
    Exemplu: int vector[100]; ( acest tablou poate stoca maxim 100 de elemente )


  • Pozitiile elementelor din vector

    • Fiecare element ocupa o anumita pozitie in memorie, identificata cu ajutorul indiciilor. Astfel, vector[0] reprezinta primul element al unui tablou cu n elemente, iar vector[n-1] este ultimul element.


  • Citirea unui vector

    • Indiferent daca tabloul este citit din fisier sau de la tastatura, el trebuie parcurs element cu element.

    Astfel se realizeaza citirea si in urmatorul exemplu, citindu-se initial dimensiunea tabloului: 

        int a[100];
    cin >> n;
    for(int i = 0 ; i < n ; i ++)
        cin >> a[i];
  • Afisarea unui tablou unidimensional

    • Afisarea se realizeaza asemanator cu citirea, inlocuind 'cin' cu 'cout' si afisand de asemenea cate un spatiu intre valori

    for(int i = 0 ; i < n ; i ++)
cout << a[i] << ' ';
  • Initializarea tablourilor unidimensionale
    • int a[10] = {0}; // initializeaza toate elementele cu 0 
int b[]={5, 10, 15, 20}; // tablou unidimensional cu 4 elemente si n=3
  • Tablouri bidimensionale

    • Aceste tablouri sunt asemanatoare cu primele, insa difera prin dimensiunea lor. Tablourile bidimensionale sunt asemenea matriciilor, avand un numar n de linii si un numar m de coloane. Fiecare matrice se parcurge cu ajutorul variabilelor i ( ptr. linii ) si j (ptr. coloane). Astfel elementul de pe linia 2, coloana 3 va fi A[1][2], numaratoarea incepand de la 0.

    • for(int j = 0 ; j < m ; j ++) { ... }
