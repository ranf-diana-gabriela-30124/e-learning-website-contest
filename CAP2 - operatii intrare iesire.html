Capitolul 2 - operatii intrare/iesire
  • operații de intrare: datele intră în program, programul citește date

    • cin = stream standard de intrare ( introducem datele de la tastatura ). cin se folosește împreună cu operatorul de extragere >>. Caracterele vor fi introduse doar dupa tastarea tastei enter

    Exemple:
    cin >> x >> y;
    cin >> a;

  • operații de ieșire: datele ies din program, programul afișează date

    • cout = stream standard de ieșire ( afisam datele pe ecran ) cout se folosește împreună cu operatorul de inserție <<

    Exemple:
    int x=3;
    cout << "Hello world";
    cout << 23 << endl;
    cout << x; // afiseaza pe ecran valoarea variabilei x
    cout << " Let's learn C++ !!";

    Output:
    Hello world23
    3 Let's learn C++ !!
Operatii de intrare/iesire cu fisiere

    Pentru început, vom analiza etapele folosirii fisierelor in C++. Asigurati-va ca aveti inclusa biblioteca 'fstream' in proiect.

  • deschiderea fisierului

    • ofstream - stream-ul pentru fluxul de iesire ( vrem sa afisam ceva in fisier ) si ifstream pentru stream-ul de intrare ( citim un continut din fisierul propriu-zis )

    Exemplu:
    ifstream fin("fisier.in");
    ofstream fout("fisier.out");

  • Citirea si scrierea in fisier

    • Exemplu:
    int x; fin >> x; fout << 2 * x;


  • Inchiderea fisierelor:

    • Exemplu:
    fin.close();
    fout.close();
