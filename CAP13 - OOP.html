Capitolul 13-Programare orientata pe obiecte

Programarea orientată pe obiect în C++ este un paradigma de programare care se bazează pe conceptul de obiecte și interacțiuni între acestea. La baza sa stă clasa, care este un tip de date definit de utilizator ce conține atât date (variabile) cât și funcții (metode) care operează asupra acestor date. În C++, variabilele membru ale unei clase pot fi declarate ca private ( pot fi accesate doar din interiorul clasei), public ( pot fi accesate din orice parte a programului ) sau protected ( depinde daca au dreptul ), controlând astfel accesul la acestea.

class Elev{
    private: 
        int varsta;
        string nume;
    public:
        Elev(){} // constructor implicit ( fara parametrii )
        ~Elev(){} // destructor implicit ( fara parametrii )

        Elev( string name )
        { nume = name; } // constructor creat de utilizator

        void setVarsta(int v) 
        { varsta = v; } // setter pentru variabila privata 

        int getVarsta()
        { return varsta; } // getter pentru variabila privata

        /// metoda ( functie ) de afisare a datelor elevului
        void afisare() { cout << "Nume: " << nume << ", Varsta: " << varsta << endl; } 
        }
  • Interfata 
    •     class Interfata {
        public:

             // metoda pur virtuala 
            // ( fara nicio implementare )
        };

        public:

            }
        };
Overload
  • Metoda overload = metoda cu acelasi nume, dar semnatura diferita
    •     class Calculator {
        public:
            int aduna(int a, int b) { return a + b; }
            int aduna(int a, int b, int c) { return a + b + c; }
            double aduna(double a, double b) { return a + b; }
        };
        
        int main() {
            Calculator calc;
            cout << calc.aduna(2, 3) << ' ' << calc.aduna(2, 3, 4) << ' ' << calc.aduna(2.5, 3.7) << endl;
            return 0;
        }

    Output:                     
5 9 6.2

Pointeri in C++
  • Definitie

    • Un pointer este o variabila care stocheaza adresa ( locul in memorie ) la care se afla o alta variabila.

  • Declarație a unui pointer către un întreg

    • int *ptr;

  • Pointerul ptr conține adresa de memorie a variabilei x

    • int x = 10; int *ptr = &x;
