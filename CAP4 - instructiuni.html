Capitolul 4 - Instructiuni in C++

  • Instructiunea if ( structura alternativa )

    • Aceasta instructiune este o structura de decizie. Daca conditia/expresia din if are valoarea 0 (este falsa), se vor executa instructiunile de pe ramura else. In caz contrar ( conditia este adevarata ), se vor executa instructiunile de pe ramura if. 


        if( expresie )
    { instructiuni; }
    else 
    { instructiuni;}

  • Instrucțiunea switch ( structura alternativa )

    • Mod de executie:
    Se evaluează conditia. Dacă valoarea acesteia este egală cu una dintre valorile constante din case, se execută instrucțiunile corespunzatoare, apoi se iasa din switch. Dacă valoarea expresiei nu este egală cu niciuna dintre valorile constante, se verifică existența clausei default; 


        switch ( Expresie )
    {
        case Constanta_1: Grup_Instructiuni_1 ; break;
        case Constanta_2: Grup_Instructiuni_2 ; break;
        …
        case Constanta_N: Grup_Instructiuni_N; break;
        default: Grup_Instructiuni_default; break;
    }

    Structurile repetitive pot fi:

  • cu număr cunoscut de pași (iterații)
  • cu număr necunoscut de pași (iterații)

    • Structurile cu nr. necunoscut de pasi pot fi cu test initial sau cu test final.

    Instructiunile repetitive sunt:

  • while :

    • număr necunoscut de pași și test inițial;

  • do...while :

    • număr necunoscut de pași și test final;

  • for :

    • număr cunoscut de pași și test inițial;

Instructiunea while
        while( conditie )
    { instructiuni }
  • Mod de executie:

    • Daca conditia este adevarata, se executa instructiunile, apoi se revine la conditia initiala si se verifica valoarea ei de adevar.
    In momentul in care conditia nu este indeplinita, se va iesi din bucla while.

Instructiunea do...while
        do{
        instructiune;
    }while( conditie );
  • Mod de executie:

    • se executa prima data instructiunea, iar apoi se verifica conditia. Se reia bucla si se execta instructiunile pana cand conditia finala este falsa.

Instructiunea for
        for (Expresie_de_Initializare ; Expresie_de_Testare ; Expresie_de_Continuare)
    {
        Instructiuni;
    }
  • Mod de executie:

    • Se evalueaza expresia de intializare si apoi se verifica valoarea de adevar a expresiei de testare.
    Se evalueaza expresia de continuare, iar apoi se verifica din nou valoarea de adevar a expresiei de testare.
    Daca aceasta este adevarata, se fac in continuare instructiunile. In caz contrar, se iese din bucla for.
