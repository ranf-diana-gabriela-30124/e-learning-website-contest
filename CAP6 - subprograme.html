Capitolul 6 - subprograme
  • Structura unui subprogram: 
    •     tip_de_date_returnat nume_subprogram ( tip_date param1, .. )
    {
        ... instructiuni; ...
        return ... ;
    }

    Componentele unui subprogram:


  • Antetul subprogramului

    • tip_de_date_returnat nume_subprogram ( tip_date param1, .. )

    Descriere: antetul functiei este format din tipul variabilei care va fi returnat la sfarsitul functiei, numele subprogramului si apoi intre paranteze lista de parametrii transmisi.


  • Corpul functiei

    • Descriere: blocul de acolade in care se gasesc instructiunile propriu-zise care trebuie realizate.

  • Observatie:

    • Exista functii care nu returneaza nimic, ci doar fac un set de instructiuni intre acolade care nu sunt accesibile in exteriorul subprogramului. Acestea sunt de tip void.


  • Cu ce ne ajuta subprogramele ?

    • Subprogramele conntribuie la o intelegere mai buna a codului, acesta fiind mai ordonat si structurat. Astfel, functia main nu mai este aglomerata inutil, realizandu-se doar apelurile la subprogramele necesare create de utilizator.
