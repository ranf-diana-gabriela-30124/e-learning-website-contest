  • Algoritm factorial 
    •     long factorial(int n) 
    { 
        if (n==0 || n==1) 
            return 1;
        return n * factorial(n-1);
    }

    Mod de executie:

    Se apeleaza in main functia: factorial(x). Daca x este diferit de 0 si 1, se trece la operatia de inmultire mediu_de_invatare x-ul curent si se apeleaza functia factorial(x-1). Acest proces are loc pana in momentul in care x este 1 sau 0 , returnandu-se astfel 1 ( 0! = 1!). Apoi algoritmul intoarce pe rand rezultatele pana se ajunge la x!, fiind returnat rezultatul final.

  • Algoritm Fibonnaci recursiv 
    •     int fib(int n) 
    {
        if (n <= 1) 
            return n;
        return fib(n-1) + fib(n-2);
    }

    Mod de executie:

    Se apeleaza in main functia: fib(x). Functia fib calculează termenul n din seria Fibonacci folosind recursivitatea. Dacă n este 0 sau 1, returneaza n. In caz contrar, returneaza suma dintre fib(n-1) și fib(n-2), continuand apelurile recursive pana ajunge la cazurile de baza.

  • Algoritmul lui Euclid recursiv ( CMMDC ) 
    •     int cmmdc(int x, int y) 
    {
        if (y == 0) return x;
        return cmmdc(y, x % y);
    }

    Mod de executie:

    Functia cmmdc calculeaza cel mai mare divizor comun (CMMD) al numerelor x și y folosind algoritmul lui Euclid. Dacă y este 0, returneaza x. In caz contrar, apeleaza recursiv cmmdc(y, x % y) pana cand y devine 0.

Definitia recursivitatii

    Recursivitatea este o tehnica de programare in care o functie se apeleaza pe ea insasi pentru a rezolva o problema, de obicei cu scopul de a simplifica solutionarea acesteia prin impartirea in subprobleme mai mici și similare. Fiecare apel recursiv adauga un nou cadru pe stiva de apeluri, consumand memorie proportional cu adancimea recursivitatii. Recursivitatea poate simplifica codul, dar poate fi ineficienta din punct de vedere al timpului și memoriei dacă subproblemele nu sunt optimizate, riscand chiar un stack overflow pentru apeluri excesive. Utilizarea unei abordări iterative poate imbunatati eficienta.
